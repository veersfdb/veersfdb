What are Data Classes ?
游릭 New feature introduced in Python 3.7
游댯 Provide concise & efficient way to define classes that only contain data, as they automatically generate several special methods, such as `__init__`, `__repr__`, and `__eq__`
游릭 Dataclasses can be customized through various options, such as specifying default values for fields, defining order for fields, and defining whether fields are mutable or not
游댯 They can also be extended and subclassed like any other class in Python.
When to use Data Classes ?
游릭 When you want to define a simple data-holding class that only contains attributes and does not need any additional behavior(methods).
游댯 Dataclasses can help you define such classes more concisely and with less boilerplate code
游릭 When you want to define a class that primarily serves as a container for data but may also to implement some basic methods, such as `__init__`,  `__repr__`  & `__eq__`
游댯 When you want to define a class that represents a structured data record or a data transfer object (DTO)
